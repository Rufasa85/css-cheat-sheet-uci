CSS Snippet Cheatsheet

My styles, let me show them to you!

Vertically centered flexbox

Vertical centering is no longer impossible!

.card{
    display:flex;
    justify-content: center;
    align-items: center;
}

Using Variables

Create CSS variable to easily reuse colors

:root{
    --primary:rgb(82, 79, 79);
    --secondary:rgb(225, 127, 247);
    --accent:white;
}

Glowy shadow

create a cool glow effect for your cards using box-shadow!

.card {
    transition: box-shadow 0.7s;
}
.card:hover, .card:hover h2{
    box-shadow:0px 0px 10px 4px var(--secondary), 
    0px 0px 10px 4px var(--secondary) inset;
}

Two-tone Gradient

Use a linear gradient to create a cool two-tone effect

pre {
    background:linear-gradient(
    120deg,
    deeppink 0%,
    deeppink 50%,
    deepskyblue 50%);
}

Overlay title on card

put the card title on top of the border!

.card h2{
    position: relative;
    bottom:18px;
    padding:5px;
}

Force Footer to Bottom

No more footers in middle of page

header,footer {
    min-height: 15vh;
}

main{
    min-height:70vh;
}
